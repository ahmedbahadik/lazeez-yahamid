# متجر لذيذ يا حامض

متجر عربي ثابت ومتجاوب لطلب طبق ورق العنب عبر واتساب، مع خياري الاستلام من المحل أو التوصيل.

## الموقع

[فتح المتجر](https://ahmedbahadik.github.io/lazeez-yahamid/)

## تعديل بيانات المتجر

السعر، رقم واتساب، وروابط التواصل موجودة في `STORE_CONFIG` أعلى ملف `app.js`.

> الطلب لا يُحفظ في قاعدة بيانات؛ يتم إرساله فعليًا عندما يضغط العميل زر واتساب في الخطوة الأخيرة. رسوم التوصيل يتم الاتفاق عليها عبر واتساب.
